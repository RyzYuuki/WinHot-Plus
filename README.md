# WinHot-Plus (v1.0.0)

Windows向けの軽量・シンプルなショートカットキー（マクロ）作成・実行ツールです。
特定のキー操作に対して、複雑なキーコンボやテキスト入力、待機処理を自由に割り当てることができます。

## ⚠️ 注意事項
- **OS環境**: 本ソフトは **Windows専用** です。
- **セキュリティソフト**: キー入力監視（フック）および擬似入力を行う性質上、アンチウイルスソフト（McAfee, Windows Defender等）に誤検知される場合があります。動作の詳細はソースコードをご確認ください。

## 🚀 主な機能

### 1. マクロの作成・編集・削除
- **新規追加**: 起動キーと実行内容を設定し、「この設定で新規追加」で有効化。
- **編集**: 「編集」ボタンから内容を書き換え、「更新（上書き）」で保存。
- **削除**: 不要になったマクロを一覧から即座に削除可能。

### 2. 特殊設定ページ
- 通常の文字キー以外（無変換、変換、Ctrl、Shift等）を起動キーに設定できる専用ページです。
- **制限事項**: 
  - 起動キーに特殊キーを設定した場合、実行内容（アクション）に特殊キーを含めることはできません。
  - 起動キーが通常キーの場合は、実行内容に特殊キーを含めることが可能です。

### 3. 特殊キー一覧
以下のキーをサポートしています：
`無変換` / `変換` / `ひらがな・カタカナ` / `半角・全角` / `左Ctrl` / `右Ctrl` / `左Shift` / `右Shift` / `Esc` / `Tab` / `Enter` / `BackSpace`

## 🛠 操作方法

### 固定ショートカット（重要）
アプリの状態を問わず、以下のキーで操作可能です。

| キー操作 | 機能 |
| :--- | :--- |
| **Ctrl + F12** | マクロ機能の **一時停止 / 再開**（ゲーム中の一時停止に便利） |
| **F12** | アプリの **完全終了**（設定は `WinHot-Plus.txt` に自動保存） |
| **[×] ボタン** | ウィンドウを隠して **タスクトレイへ格納** |

> ※タスクトレイのアイコンをダブルクリックすると、設定画面が再表示されます。
